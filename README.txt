# Space wars

* Open the VS project file, found inside "MyGame" folder -> "MyGame.sln"
* Requires Microsoft DirectX 2010
* Check "Include" and "Library" directories 
* Build & Run project
* LEVEL 1
	* A and D keys to move 
	* SPACE bar to shoot
* LEVEL 2
	* WASD to move
	* SPACE bar to shoot
* Don't collide with enemy spaceship bullets and asteroid
* Shoot enemy spaceships and asteroids to win
